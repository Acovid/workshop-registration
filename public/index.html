IBM Cloud for Developers Workshop

September 27, 2018 at 9.00 - 16.00

Ljubljana


This workshop provides a review of key concepts related to IBM Cloud Platform (CP) - formerly known as Bluemix. It is structured as a series of lectures accompanied by demonstrations and hands-on labs. Completing this course will give you experience with IBM CP and show you how to build and deploy applications for the cloud that are scalable, reliable and secure. This one day workshop is also a first step in preparation for the certification exam “IBM Cloud Platform Application Development V2”. Other steps and materials necessary to earn the above mentioned certificate will be shared at this workshop.

Content

Part 1: Create applications on IBM Cloud.
Understand cloud service models and benefits, IBM Cloud architecture, best practices in cloud application design.

Part 2: Enhance applications with CP services and security.
Learn about different services available in the CP catalog and steps that you can take to secure your applications, credentials for services, and application data.

Part 3: Continuous delivery and management of cloud apps.
Learn how CP supports DevOps by using Continuous Delivery toolchains. Learn about techniques to monitor applications, manage the scaling of applications, and debug application issues.

Agenda

  1. IBM Cloud models and capabilities - lecture
  2. App development, deployment and best practices in cloud - lecture
  3. CP management and Command Line Interface (CLI) - lecture
  4. Use CLI to deploy and manage apps on CP - lab
  5. Enhance apps with CP messaging and Object storage - lecture
  6. Data services and databases on CP – lecture
  7. Add and use Object Storage service – lab
  8. Setup and use Cloudant, a NoSQL database - lab
  9. Source code management, continuous delivery and pipeline – lecture
  10. App scaling and monitoring, load testing – lecture
  11. Security – lecture
  12. Optional labs:
    • Create and use a toolchain
    • Code source control management with Git
    • Manage continuous delivery and integration
